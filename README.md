# Rock-Paper-Scissor

A simple and interactive Rock Paper Scissors game built using **HTML**, **CSS**, and **JavaScript**. The game allows users to play against the computer with real-time score tracking and result updates.

##  Features

*  Rock,Paper,Scissors gameplay
*  Random computer choice generation
*  Live score tracking
*  Win/Lose/Draw result display
*  Dynamic message colors based on game outcome
*  Responsive and user-friendly interface

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Play

1. Click on Rock, Paper, or Scissors.
2. The computer will randomly select its move.
3. The winner is determined according to the rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. Scores are updated automatically.
5. The result message displays whether you won, lost, or drew the round.
